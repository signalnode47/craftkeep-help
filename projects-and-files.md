# Projects And Files

## Project Folders

Each Craftkeep project points to a folder on your computer. That folder is the source of truth for the project.

Craftkeep is local-first, so your data is not locked inside a remote account. The app reads from and writes to files in the selected folder.

## Choosing A Folder

When creating a project, choose the folder where that project should live. A good pattern is:

```text
Craftkeep/
  Project Name/
```

For now, keep each Craftkeep project in its own folder. Avoid pointing multiple active projects to the same folder unless you are intentionally testing.

## Backups

Back up the project folder like any other important creative work:

- Time Machine
- External drive
- iCloud Drive, Dropbox, Google Drive, or similar folder sync
- Manual copy to an archive folder

Craftkeep does not replace backups.

## Deleting Or Hiding Projects

When you remove or hide a project in Craftkeep, the files on disk are not deleted. The app only removes that project from the active project list.

If you want to delete project files permanently, do that yourself in Finder after confirming you no longer need them.
