# Getting Started

## Open Craftkeep

When Craftkeep opens for the first time, it should show the bundled Mock Project. This gives you a safe demo workspace to explore before creating your own project.

## Create A Project

1. Click the `+` button in the top tab bar.
2. Enter a project name.
3. Click `Create project`.
4. Choose a folder in Finder.

Craftkeep will use that folder as the home for the project files it creates and manages.

## Switch Projects

Projects appear as tabs across the top of the app. Click a tab to switch projects.

To hide a project from the tab bar, click the `x` on the tab. Hiding a project removes it from the app list, but it does not delete files from disk.

## Use The Main Sidebar

The sidebar is the main navigation area:

- `Home`: project overview and current priorities
- `Project Hub`: links, EPK, press, brand, and bio details
- `Releases`: launch and release planning
- `Touring`: show and travel planning
- `Tasks`: action items
- `Goals`: progress tracking
- `Notes`: freeform notes and writing
- `Content`: content calendar and ideas
- `Network`: contacts and outreach
- `Campaigns`: marketing and promotion
- `Analytics`: performance snapshots

## Use Mock Project

Mock Project is demo data that ships with Craftkeep. Open it from the Help menu when you want to explore without touching a real project.
