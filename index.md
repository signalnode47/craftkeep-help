# Craftkeep Help

Craftkeep is a local-first project manager for independent creatives. It keeps your work in plain files on your computer, with the app acting as a focused workspace for planning, releases, tasks, contacts, notes, and creative operations.

This documentation is the temporary home for Craftkeep help while the public website is not live yet.

## Quick Answers

### Where is my data stored?

Your data lives in the project folder you choose when creating a project. Craftkeep reads and writes local markdown/data files inside that folder.

### What is the Mock Project?

Mock Project is bundled demo data. Use it to explore Craftkeep without risking real project files.

### Can I use Craftkeep for more than music?

Not fully yet. Craftkeep is currently focused on music projects, but broader creative support is part of the future plan.

### Does Craftkeep sync to the cloud?

Not yet. The current alpha is local-first. You can back up your project folder with your normal backup tools, external drive, or cloud folder provider.

### Is there a mobile app?

Not yet. A mobile app is part of the future plan, including sync so you can access Craftkeep across devices.
