# Settings

Settings are organized around the parts of Craftkeep you are most likely to adjust during alpha.

## General

### Currency

Sets the currency used for budgets and financial displays.

### Official Documentation

Opens this help documentation in your browser.

## Projects

Use this area to:

- View projects known to Craftkeep
- Switch active project
- Rename a project in the app
- Remove a project from the app list
- Change the active project folder
- Open Mock Project

Removing a project from the app does not delete files from disk.

## Appearance

### Theme

Switches between light, dark, and system appearance.

### Zoom

Scales the app workspace and interface.

### Font Size

Adjusts the base UI font size.

### Accent Color

Changes active control and selected-state color. Use `Reset` to return to the default Craftkeep accent.

### Sidebar Default

Chooses whether the sidebar starts expanded or collapsed.

## Dashboard

### Pinned Items

Clears items pinned to Home.

## Files And Data

### Export

Downloads a JSON snapshot of the current project.

### Import

Merges a previous export into the current project.

## Advanced

Advanced settings are for alpha testing and debugging. Use them carefully.
